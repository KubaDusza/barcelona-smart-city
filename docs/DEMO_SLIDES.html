Deployment & Ops

SETUP
bash aws/setup.sh
tables, roles, storage
bash aws/deploy.sh all
deploy collectors + MCP server
python3 aws/scripts/load_gtfs.py
load transit stops (one-time)
python3 aws/scripts/verify_data.py
confirm data is flowing
RUNNING
EventBridge
triggers collectors on schedule
DynamoDB
stores data, auto-expires after 30 days
IAM
access control per data vertical
bash aws/pause.sh
stop data collection
bash aws/teardown.sh
delete everything

4. Start a new chat — tools auto-approve

“I’m going for a run in Barcelona tomorrow morning. Will the air quality and pollen be okay?”
“I want to take my kids to Barceloneta this afternoon — is the UV too strong? What SPF do we need?”
